# papers/

Source PDFs for the climate-paper search index. PDFs are gitignored (large, easily re-downloaded), so this folder is empty until you add your own.

Starting corpus used for this project:
- IPCC AR6 Synthesis Report (Summary for Policymakers): https://www.ipcc.ch/report/ar6/syr/downloads/report/IPCC_AR6_SYR_SPM.pdf
- IPCC SR15 (Special Report on Global Warming of 1.5°C)
- IPCC AR6 Synthesis Report (full volume)

After adding PDFs here, run:
```bash
uv run python ingest.py
```
